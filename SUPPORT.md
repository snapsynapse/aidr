# Support

AIDR is a small open specification and reference linter.

## Questions and proposals

Use GitHub issues for:

- questions about adopting AIDR
- proposed template or documentation changes
- linter bugs
- compatibility requests with related decision-record formats

## Security reports

Do not open public issues for exploitable vulnerabilities. Follow [SECURITY.md](SECURITY.md).

## Maintainer scope

The maintainer can clarify the AIDR format and reference linter. The maintainer cannot provide project-specific legal, compliance, security, or audit approval for records in other repositories.
